# Bridging Brushes Web

Frontend web application for Bridging Brushes, built with React and designed to showcase custom design services and digital brand presence.

## Overview

This project is a React-based web application developed to present the Bridging Brushes brand online through a clean interface, structured pages, and responsive user experience.

## Features

- Responsive React frontend
- Component-based architecture
- Modern UI for service and brand presentation
- Structured static pages and reusable sections
- Ready for hosting and deployment

## Tech Stack

- React
- JavaScript
- CSS
- Create React App

## Project Structure

- `src/` – components, pages, styles, and frontend logic
- `public/` – static assets
- `package.json` – dependencies and scripts
